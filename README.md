GeiDeLei henceforth known as "the app" is provided as is. The user interface is intended to be self explanatory and very simplistic. 

The app relies on third party services that are freely available. In this sense the app acts as a browser that comfortably lets you use these web services and interpret the results in a practical way. It also means that these services may at any point in time be changed, removed or updated as they are not part of the app itself. 

The app uses the System Settings for accessibility to select voice quality. This means that you can download the enhanced voices for better quality if you want to have a better user experience.

Using the "Do a flip" setting allows for self dialogue and easy learning of short phrases, words and new language.

As app developer the goal is to keep the app connected to the best available free services to provide translation at low cost.
